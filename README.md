# 🪙 Alke Wallet

Proyecto integrador desarrollado durante un bootcamp de desarrollo SENCE FullStack Java.

Este repositorio contiene la evolución completa de **Alke Wallet**, una aplicación de billetera digital construida de manera incremental a través de las distintas evaluaciones del programa.

Cada módulo incorpora nuevas tecnologías, funcionalidades y mejoras arquitectónicas, permitiendo visualizar el progreso del proyecto mediante el historial de commits.

---

## 📖 Descripción

Alke Wallet es una aplicación web que simula una billetera digital para la administración de fondos y transacciones.

El proyecto se desarrolla de forma progresiva, donde cada evaluación del bootcamp agrega nuevos componentes al sistema.

Entre las funcionalidades que se incorporarán a lo largo del desarrollo se encuentran:

* Administración de saldo.
* Depósitos y transferencias.
* Historial de movimientos.
* Persistencia de datos.

## 🐳 Entorno de Desarrollo

El proyecto utiliza Docker únicamente para facilitar el entorno de desarrollo local y asegurar una configuración consistente durante las distintas evaluaciones del bootcamp.

La configuración se encuentra en:

```text
docker/
```

### ▶️ Iniciar el entorno

```bash
./start.sh
```

### ⏹️ Detener el entorno

```bash
./stop.sh
```

### 🌐 Acceso

Una vez iniciado el entorno, la aplicación estará disponible en:

```text
http://localhost:8080
```

> Docker se utiliza exclusivamente con fines de desarrollo y pruebas locales. No forma parte de una estrategia de despliegue o entorno productivo.
